<h1>🌱 EcoCycle Industry</h1>

<p>Uma página web sobre sustentabilidade industrial e reciclagem de resíduos industriais.</p>

<p>O projeto foi desenvolvido com o objetivo de mostrar como a indústria pode reduzir impactos ambientais através da reciclagem, reutilização de materiais e práticas sustentáveis.</p>

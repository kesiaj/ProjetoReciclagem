## 📖 Sobre o Projeto

O EcoCycle Industry é um site criado com o objetivo de conscientizar sobre a importância da reciclagem industrial e das práticas sustentáveis dentro das empresas.

Através de informações, ações práticas e conteúdo educativo, o projeto busca mostrar como a reutilização de materiais e a redução de resíduos contribuem para a preservação do meio ambiente e para uma indústria mais sustentável.
